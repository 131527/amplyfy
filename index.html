Tecnologia de Segurança Utilizada: HTTPS (TLS)

Este site foi publicado utilizando AWS Amplify, que fornece conexao segura através do protocolo HTTPS, utilizando criptografia TLS para proteger os dados transmitidos entre o usuário e o servidor.

O uso de HTTPS garante:

  • Confidencialidade dos dados
  • Integridade das informacoes
  • Autenticacao do servidor
